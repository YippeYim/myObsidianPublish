เราใช้ Rain Sensor ตรวจจับความชื้นภายนอก เชื่อมต่อกับบอร์ด STM32 Nucleo เพื่อควบคุม มอเตอร์ ให้ปิดม่านทันทีเมื่อฝนตก พร้อมทั้งใช้ Node-RED เก็บข้อมูลสภาพอากาศลงฐานข้อมูล

Raspberry Pi ทำหน้าที่เป็น IoT Gateway ส่งข้อมูลขึ้น Cloud เพื่อมอนิเตอร์สถานะ
